# Atividade Avaliativa: Estruturas de Árvores

Este repositório contém o trabalho prático e teórico sobre estruturas avançadas de árvores e suas operações de balanceamento, desenvolvido para a disciplina de Estrutura de Dados.

## 📋 Conteúdo do Trabalho

* **Parte 1 – Tipos de Árvores:** Análise conceitual sobre Árvores AVL, Rubro-Negras e N-árias.
* **Parte 2 – Operações em Árvores:** Explicação prática de Rotações (Simples e Duplas) e Espelhamento.
* **Parte 3 – Aplicação Prática:** Estudo de caso sobre o uso de Árvores B/B+ na indexação de Bancos de Dados Relacionais.
* **Parte 4 – Comparação entre Estruturas:** Tabela comparativa detalhando vantagens, desvantagens e complexidade ($O$) de cada árvore.
